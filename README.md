# Kbank PC Prototype

케이뱅크 PC 웹사이트 프로토타입입니다.

## 배포

이 프로젝트는 GitHub Pages를 통해 배포할 수 있습니다.

### 배포 방법

1. GitHub 저장소의 Settings로 이동
2. Pages 메뉴 선택
3. Source에서 "Deploy from a branch" 선택
4. Branch를 "main"으로 선택하고 "/ (root)" 선택
5. Save 버튼 클릭

배포 후 몇 분 후에 다음 URL에서 접근할 수 있습니다:
`https://jieunbang.github.io/KBANK_PCweb_prototype`

## 로컬 실행

```bash
# Python 3를 사용한 간단한 HTTP 서버
python3 -m http.server 8080
```

브라우저에서 `http://localhost:8080` 접속

## 주요 기능

- 반응형 헤더 네비게이션
- 섹션별 스크롤 애니메이션
- 스토리 카드 슬라이더
- 푸터 스크롤 애니메이션
